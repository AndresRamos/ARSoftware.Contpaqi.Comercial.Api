# AR Software - CONTPAQi Comercial API

## Haz interfaz con el sistema de CONTPAQi Comercial Premium a travez de un servicio web en la nube.

**[AR Software - CONTPAQi Comercial API](https://www.arsoft.net/contpaqi-comercial-api)** permite la interfaz entre tu sistema y el sistema de CONTPAQi Comercial Premium. Con ella podras crear, modificar, y consultar los diferentes documentos y catalogos desde tus aplicaciones nube o aplicaciones desktop que existan fuera de la red local. Tendras acceso al sistema desde cualquier lugar. CONTPAQi Comercial Premium en la nube!

Esta API esta diseñada para permitirte realizar cualquier operacion disponbile por el SDK de CONTPAQi Comercial Premium. Con ella podras seguir gozando de los beneficios del sistema como lo es la creacion de facturas de forma masiva y el timbrado ilimitado. Solamente consulta la documentacion de la operacion que deseas realizar, genera la estructura JSON, y mandala a la API. Ya no tendras que preocuparte de aprender los pequenos detalles y requerimientos necesarios para implementar el SDK de CONTPAQi Comercial.
